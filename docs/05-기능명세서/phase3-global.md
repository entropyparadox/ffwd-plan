# Phase 3: Festival & Global (Month 13~24)

> "Enter New Markets with an Edge. Elevate Every Signal."

## 글로벌 확장 × 프리미엄화

Phase 3는 두 가지 전략이 동시에 작동하는 단계다.

- **글로벌 확장** — "Enter New Markets with an Edge." 일본·동남아·글로벌로 진출하되, 한국 시네마라는 문화적 무기를 앞세운다.
- **프리미엄화** — "Elevate Every Signal." 영화제, 제작지원, 라이브 스트리밍 등 모든 터치포인트에서 프리미엄 시그널을 방출한다.

**핵심 질문**: "FFWD가 5개국에 진출했을 때, 각국에서 어떤 한 단어로 기억될 것인가?" → "Discovery." 발견. 다음 봉준호가 발견되는 곳.

## 8개 글로벌 기능

| # | 기능 | 우선순위 | 설명 |
|---|---|---|---|
| F-26 | 영화제 시스템 | P0 | 노미네이션, 온라인 투표, 실시간 시상식 스트리밍 |
| F-27 | 제작지원 관리 | P0 | 지원 신청 → 심사 → 선정 → 제작비 집행 → 결과물 등록 |
| F-28 | 다국어 UI | P1 | 영어, 일본어, 중국어, 스페인어 UI |
| F-29 | 글로벌 결제 | P1 | 국가별 결제 수단 (Stripe 글로벌) |
| F-30 | API 공개 | P2 | 외부 서비스 연동용 공개 API |
| F-31 | 국가별 파트너 포털 | P2 | 각국 대학과의 작품 등록 시스템 |

### 시장 기회 개척

Phase 3의 8개 기능 중 5개(F-28~F-31, F-33)가 시장 확장 기능이다. 기존 시장에서 성공한 모델을 새로운 시장에 **Edge(우위)**와 함께 진입시키는 것이 핵심이다.

> FFWD의 Edge: 이정재 브랜드 × 한국 시네마 문화 × 검증된 큐레이션 시스템
