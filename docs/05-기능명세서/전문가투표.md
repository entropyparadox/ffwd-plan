# 전문가 투표 시스템

> "Don't Become the Best. Become the Only."

## 선택 설계 (Choice Architecture)

전문가 투표 시스템은 FFWD의 핵심 브랜드 도구다. 누가 평가하고, 어떤 기준으로 평가하고, 결과가 어떻게 표시되는지 — 이 모든 것이 "FFWD 작품은 믿고 본다"는 브랜드 신뢰를 설계하는 구조다.

> 핵심 프레이밍: Less "누가 이겼나" → More "왜 이 작품이 특별한가"

## 전문가 등급

```
[전문가 등급]
├── Tier 1: 영화 감독/배우 (이정재 네트워크) — 가중치 ×3
├── Tier 2: 영화 평론가 (인증) — 가중치 ×2
├── Tier 3: 영화 유튜버 (구독자 10만+) — 가중치 ×1.5
└── Tier 4: 업계 종사자 (인증) — 가중치 ×1
```

> 전문가 등급 체계는 "이 사람의 평가를 왜 신뢰해야 하는가?"를 구조적으로 증명한다.

## 평가 항목

| 항목 | 비중 |
|---|---|
| 연출력 | 25% |
| 스토리텔링 | 25% |
| 연기력 | 20% |
| 촬영/미술 | 15% |
| 음악/사운드 | 15% |

## 통합 랭킹 산출

```
[통합 랭킹]
├── 전문가 점수 (40%) — 가중 평균
├── 대중 점수 (30%) — 5점 만점 평균
└── 시청 지표 (30%)
     ├── 조회수
     ├── 완주율
     ├── 열람 전환율 (무료→유료)
     └── 재시청률
```

### 구조화된 집단 지성

"민주주의적 접근": 소수 게이트키퍼가 아닌, 구조화된 집단 지성으로 작품을 발굴. 전문가의 전문성 + 대중의 공감 + 데이터의 객관성을 균형 있게 결합.

> 이 3축 구조는 전문가(품질), 대중(문화적 공감), 데이터(시장 검증)가 하나의 랭킹에 정렬되는 시스템이다.
