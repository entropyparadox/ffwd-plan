# 자동 자막 파이프라인

> "Borders end where stories begin."

## Language Bridge — 언어 장벽 제거

자동 자막은 단순한 기술 기능이 아니라 FFWD의 Language Bridge 시스템이다. 한국 대학 졸업작품이 5개국어로 번역되는 순간, 그것은 "한국 영화"가 아니라 **"글로벌 시네마"**가 된다. 언어 장벽을 제거하는 것은 곧 시장 장벽을 제거하는 것이다.

## 파이프라인 흐름

```
[원본 영상 업로드]
  → [Whisper Large v3] 한국어 STT (정확도 95%+)
  → [SRT 자동 생성] + [타임스탬프 정렬]
  → [DeepL API] 5개국어 번역
     ├── 영어 (EN)
     ├── 일본어 (JA)
     ├── 중국어 간체 (ZH-CN)
     ├── 중국어 번체 (ZH-TW)
     └── 스페인어 (ES)
  → [QA 검수] 인력 최종 확인 (선택적)
  → [자막 배포] 플레이어에 다국어 자막 적용
```

## 기술 스택

| 단계 | 기술 | 비용 |
|---|---|---|
| 음성 인식 | OpenAI Whisper Large v3 | $0.006/분 |
| 번역 | DeepL API Pro | $20/100만 글자 |
| 자막 포맷 | SRT/WebVTT | — |
| 검수 | 인력 QA (선택) | ₩15,000/시간 |
| 저장 | Cloudflare R2 | $0.015/GB |

## 품질 보증

- Whisper v3: 한국어 WER(Word Error Rate) ~5%
- DeepL: 한→영 번역 품질 업계 최고 수준
- QA 선택제: 창작자가 자막 직접 수정 가능한 에디터 제공
- 커뮤니티 번역: 팬 자발 번역 시스템 (Year 2+)

::: danger Whisper 환각(Hallucination) 경고
최근 연구에 따르면 Whisper의 자동 음성 인식에서 **약 80%의 전사(transcription)에 환각(fabricated) 콘텐츠가 포함**될 수 있다는 보고가 있습니다. 이는 존재하지 않는 대사를 생성하거나, 실제 대사를 왜곡하는 형태로 나타납니다.

FFWD에서는 이 문제를 해결하기 위해 **다단계 QA 프로세스**를 운영합니다. 자세한 내용은 [자막 품질 보증 (QA) 프로세스](/05-기능명세서/자막QA) 페이지를 참조하세요.
:::

## 시나리오 기반 설계

자동 자막 파이프라인은 세 가지 사용자 시나리오를 동시에 해결한다:

1. **창작자**: "내 작품이 자동으로 5개국어로 번역된다" → 글로벌 노출 장벽 제거
2. **해외 관객**: "한국 졸업작품을 모국어 자막으로 볼 수 있다" → 콘텐츠 접근성 확보
3. **영화제 바이어**: "이미 다국어 자막이 준비된 작품 풀" → IP 라이선싱 즉시 가능
