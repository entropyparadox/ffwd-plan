# Phase 1: MVP (Month 1~6)

> "Build the Next Hottest Thing."

## 카테고리 창조

Phase 1은 카테고리 창조 단계다. 새로운 카테고리를 만들고, 첫 인상을 설정하고, 시장에 존재를 선언하는 단계. "완벽한 제품"이 아니라 **"시장이 반응하는 최소한의 선언"**을 목표로 한다.

**핵심 질문**: "6개월 후, 시장이 FFWD를 어떤 한 문장으로 기억할 것인가?" → "대학 졸업작품을 3분 단위로 볼 수 있는 유일한 플랫폼."

## 15개 핵심 기능

| # | 기능 | 우선순위 | 설명 |
|---|---|---|---|
| F-01 | 작품 등록 | P0 | 영상 업로드 + 메타데이터(제목, 학교, 제작진, 시놉시스) |
| F-02 | 자동 에피소드 분할 | P0 | 90분 영상 → 3분 단위 에피소드 자동 분할 (수동 조정 가능) |
| F-03 | 자동 자막 생성 | P0 | Whisper 기반 한국어 STT → SRT 자동 생성 |
| F-04 | 자동 번역 | P1 | 한국어 자막 → 영어/일본어/중국어 자동 번역 |
| F-05 | 웹 플레이어 | P0 | HLS 적응형 스트리밍, 자막 ON/OFF, 화질 선택 |
| F-06 | 무료/유료 열람 | P0 | 첫 3화 무료, 이후 에피소드별 결제 (₩300~500) |
| F-07 | 결제 시스템 | P0 | Toss Payments 연동, 코인 충전 모델 |
| F-08 | 회원가입/로그인 | P0 | 이메일 + 카카오/네이버/구글 소셜 로그인 |
| F-09 | 작품 상세 페이지 | P0 | 시놉시스, 제작진, 에피소드 목록, 평점, 리뷰 |
| F-10 | 탐색/검색 | P1 | 장르, 학교, 연도별 필터 + 풀텍스트 검색 |
| F-11 | 좋아요/북마크 | P1 | 작품별 좋아요 + 내 보관함 |
| F-12 | 대중 투표 | P1 | 작품별 5점 만점 평가 + 댓글 |
| F-13 | 시청 기록 | P1 | 이어보기, 시청 히스토리 |
| F-14 | 창작자 대시보드 | P1 | 조회수, 열람수, 수익, 투표 현황 |
| F-15 | 관리자 대시보드 | P0 | 작품 승인/거절, 사용자 관리, 수익 정산 |

### MVP 목표 — 시장 차별성 확립

6개월 내 50개교 파트너십, 500편 등록, MAU 10,000 달성

> 이 숫자가 중요한 이유: 50개교는 전국 200개교의 25%. 1/4의 시장을 선점하면 "졸업작품 = FFWD"라는 카테고리 연상이 형성된다.
