# 자막 품질 보증 (QA) 프로세스

> Whisper AI의 자막에는 **환각(hallucination)**이 포함될 수 있다. 자동화만으로는 부족하다.

## Whisper 환각 문제

### 현황

| 항목 | 수치 | 출처 |
|------|------|------|
| 환각 발생률 | **10개 중 8개** 전사에 환각 포함 | UMich 연구 / DIYAI |
| 한국어 WER (단어 오류율) | ~5~8% | OpenAI |
| 영어 WER | ~4~5% | OpenAI |
| Google Chirp 2 WER | ~3.5% | DIYAI 벤치마크 |

### 환각 유형

| 유형 | 예시 | 위험도 |
|------|------|--------|
| 허구 내용 삽입 | 무음 구간에 존재하지 않는 대사 생성 | 🔴 치명적 |
| 반복 루프 | 같은 문장 무한 반복 | 🟡 눈에 띔 |
| 언어 전환 | 한국어 중 갑자기 영어 출력 | 🟡 혼란 |
| 고유명사 오류 | 인물/장소명 변형 | 🟠 맥락 왜곡 |
| 감정/톤 왜곡 | 분노 대사를 평서문으로 전사 | 🟡 뉘앙스 손실 |

::: danger ⚠️ 시네마틱 콘텐츠에서의 영향
영화에서 대사 하나가 틀리면 **감독의 의도가 왜곡**됩니다. 일반 UGC와 달리, 시네마틱 숏폼의 자막 정확도는 **브랜드 신뢰와 직결**됩니다.
:::

## 다단계 QA 파이프라인

```
[Stage 1: 자동 전사]
    Whisper Large v3 → 한국어 STT
    비용: $0.006/분 (에피소드당 ₩24)
    ↓
[Stage 2: 자동 보정]
    - 반복 루프 자동 감지/제거
    - 타임스탬프 정렬 보정
    - 환각 패턴 필터링 (규칙 기반)
    비용: 자동화 (추가 비용 없음)
    ↓
[Stage 3: 창작자 셀프 리뷰]
    - 창작자에게 자막 에디터 제공
    - 원본 대사와 비교 확인
    - 직접 수정 가능
    비용: ₩0 (창작자 자발적 참여)
    ↓
[Stage 4: 커뮤니티 리뷰] (Year 2+)
    - 팬 자발 번역/검수 시스템
    - 언어별 리뷰어 배지
    비용: ₩0 (커뮤니티 기여)
    ↓
[Stage 5: 전문 QA] (프리미엄 콘텐츠만)
    - COMMISSION/SIGNATURE 작품 한정
    - 전문 자막 검수자
    비용: ₩15,000/시간 (에피소드당 ~₩7,500)
```

## 비용 비교

| 방식 | 에피소드당 비용 | 정확도 | 적용 범위 |
|------|--------------|--------|----------|
| Whisper만 (Stage 1) | ₩24 | ~92~95% | — |
| + 자동 보정 (Stage 2) | ₩24 | ~95~97% | 전체 |
| + 창작자 리뷰 (Stage 3) | ₩24 | ~97~99% | 전체 |
| + 전문 QA (Stage 5) | ₩7,524 | ~99%+ | COMMISSION만 |
| 전체 인력 QA (비교용) | ₩30,000+ | ~99%+ | — |

## 품질 지표 (SLA)

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 한국어 전사 정확도 | 95%+ | 랜덤 샘플 인력 검증 |
| 번역 의미 보존율 | 90%+ | 이중 언어 검수자 평가 |
| 환각 발생률 | 2% 이하 | Stage 2 자동 감지 로그 |
| 자막 동기화 오차 | ±0.5초 이내 | 자동 측정 |
| 창작자 자막 만족도 | 4.0/5.0+ | 업로드 후 설문 |
