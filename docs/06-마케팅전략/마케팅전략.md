# 마케팅 전략

> "Build the Next Hottest Thing."

FFWD의 마케팅은 "광고"가 아니라 **"카테고리 창조"**다. 시네마틱 숏폼이라는 새로운 영토를 정의하고, 그 영토의 주인이 되는 것.

## 마케팅 3대 원칙

### 1. 문제 정의 우선 — 고통을 먼저 명명하라

일반 마케팅은 "우리가 뭘 잘하는지"를 말한다. FFWD의 마케팅은 **"당신이 뭘 놓치고 있는지"**를 말한다.

| 대상 | 문제 (고통) | FFWD의 답 |
|---|---|---|
| 영화 감독 | "숏폼에서는 내 비전을 펼칠 수 없어" | "3분 안에 당신의 세계관을 완성하세요" |
| 시네마토그래퍼 | "아무도 내 조명을 신경 쓰지 않아" | "여기서는 빛 하나가 이야기입니다" |
| 관객 | "의미 있는 숏폼을 찾을 수 없어" | "매 프레임이 감독의 선택입니다" |
| 대학 졸업생 | "졸업작품을 아무도 안 봐줘" | "당신의 첫 번째 관객이 100만 명일 수 있습니다" |
| 투자자 | "숏폼은 수익 모델이 불분명해" | "프리미엄 숏폼은 프리미엄 가격을 커맨드합니다" |

### 2. 상위 가치 이동 — 제품이 아닌 의미를 팔아라

| 일반 마케팅 | FFWD |
|---|---|
| "영화인을 위한 숏폼 플랫폼입니다" | "시네마의 다음 캔버스" |
| "좋은 영상을 올릴 수 있습니다" | "짧은 형식이 예술이 되는 곳" |
| "크리에이터에게 수익을 드립니다" | "당신의 프레임에 정당한 가치를" |
| "졸업작품을 공개할 수 있습니다" | "봉준호도 졸업작품에서 시작했습니다" |

## 유튜버 파트너십 네트워크

### 전체 도달력

| 메트릭 | 수치 |
|---|---|
| 파트너 채널 수 | 20+ |
| 총 구독자 합계 | 1,500만+ |
| 예상 1회 캠페인 노출 | 500만~1,000만 views |
| 코어 타겟 도달 (20-30대 영화팬) | 300만+ |

### S-Tier: 메가 인플루언서

- **지무비 (G Movie)** — 395만 구독자. 론칭 앰배서더. 단일 영상 100만+ 조회 기대.
- **조승연 (탐구생활)** — 280만 구독자. "왜 숏폼 시네마인가" 사회적 담론 확산.
- **고몽 (Gomong)** — 254만 구독자. 졸업작 명장면 해설.

### A-Tier: 감성 브랜딩

- **김시선** — 200만. 에세이형 리뷰.
- **배우는 사람** — 120만+. 신인 감독 인터뷰.

### B-Tier: 기획 콘텐츠

| 채널 | 구독자 | 협업 포맷 |
|---|---|---|

| 띵잘 ⭐ | ~70만 | "숨겨진 보석" 시리즈 (최적 파트너) |

| 소개해주는 남자 | ~60만 | 신인 발굴 |
