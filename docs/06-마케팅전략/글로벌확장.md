# 글로벌 확장

> "Enter New Markets with an Edge."

FFWD의 글로벌 확장은 "번역"이 아니라 **"문화 동기화"**다. 각 국가의 영화 문화와 공명하며, "시네마틱 숏폼"이라는 카테고리를 글로벌 표준으로 만든다.

## 확장 원칙

### 1. Culture Sync Logic — 문화적 맥락과 동기화

| 일반 글로벌 확장 | FFWD 글로벌 확장 |
|---|---|
| "한국 서비스를 번역해서 출시" | "각 나라의 영화 문화에 동기화" |
| "글로벌 캠페인 일괄 집행" | "로컬 문제를 먼저 명명" |
| "같은 기능, 다른 언어" | "같은 철학, 다른 표현" |

### 2. Heritage Capsule — 각국 영화 유산을 레버리지

| 국가 | Heritage Capsule |
|---|---|
| 🇰🇷 한국 | 봉준호, 박찬욱, 이정재 — "K-Cinema의 다음 세대" |
| 🇯🇵 일본 | 구로사와, 고레에다 — "자주영화(自主映画)의 현대적 계승" |
| 🇹🇼 대만 | 허우샤오시엔, 에드워드 양 — "뉴 타이완 시네마의 숏폼 실험" |
| 🇮🇳 인도 | 사티야지트 레이, 파랄렐 시네마 — "볼리우드 너머의 인도 영화" |
| 🇫🇷 프랑스 | 누벨바그 — "고다르가 오늘 영화를 만든다면, 3분짜리를 만들었을 것" |
| 🇺🇸 미국 | 선댄스, 인디 — "할리우드 밖의 목소리" |

## 확장 로드맵

```
[Phase 1: ORIGIN] Year 1~2 — 카테고리 창조
  한국 시장 검증 + K-Cinema 유산 활용

  → [Phase 2: MOMENTUM] Year 3 — 아시아 확장
     ├── 🇯🇵 일본 — 자주영화 문화와 동기화
     ├── 🇹🇼 대만 — 뉴웨이브 시네마 연결
     ├── 🇹🇭 태국, 🇮🇩 인도네시아 — ASEAN 영화학교 네트워크
     └── "Enter New Markets with an Edge."

  → [Phase 3: AUTHORITY] Year 4~5 — 글로벌 프리미엄화
     ├── 🇫🇷 유럽 — 칸·베를린 공식 파트너십
     ├── 🇺🇸 북미 — USC/NYU/AFI 네트워크
     └── "Elevate Every Signal."

  → [Phase 4: GLOBAL] Year 5+
     └── 100개국+ 글로벌 포털
```

## 비전: "영화계의 GitHub"

GitHub이 전 세계 개발자의 코드를 연결했듯, FFWD는 전 세계 영화인의 작품을 연결한다. 각국 영화학교가 FFWD에 작품을 올리고, 글로벌 관객과 전문가가 발굴하는 생태계. 국경은 자막이 해결하고, 품질은 큐레이션이 보장한다.
