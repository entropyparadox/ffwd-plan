# FFWD 아이디어 랩

> "Press Play on the Future." — 아이디어도 마찬가지다. 작지만 강렬한 것들.
>
> A24, NEON, @filmoment 벤치마크에서 영감을 받아 FFWD만의 확장 프로그램을 설계한다.

---

## 1. FFWD 팟캐스트 — "프레임 너머의 대화"

> A24의 팟캐스트("Nervous Laughter with Joaquin Phoenix & Ari Aster")에서 영감. 그러나 FFWD는 **기성 감독이 아닌 차세대 감독**과 거장이 만나는 유일한 대화 공간.

### 콘셉트

**"프레임 너머 (Beyond the Frame)"** — 영화는 스크린 안에서 끝나지 않는다. 프레임 너머의 이야기가 더 깊다.

이정재 감독을 호스트로, 배우·평론가·졸업생 감독이 함께하는 시네마 대담 프로그램. 핵심은 **"세대를 잇는 대화"** — 거장과 신인이 같은 테이블에 앉는다.

### 프로그램 라인업

| 시리즈 | 포맷 | 출연 | 주기 | 설명 |
|--------|------|------|------|------|
| **감독의 밤** | 오디오 + 영상(60분) | 이정재 × 졸업생 감독 | 격주 | 졸업 작품 비하인드를 감독이 직접 풀어내는 1:1 대담 |
| **배우의 눈** | 오디오 + 영상(45분) | 초청 배우 × 졸업생 감독 | 월 1회 | 배우가 졸업 작품을 보고 "이 역할을 받았다면 어떻게 연기했을까" 대화 |
| **평론가의 돋보기** | 오디오(30분) | 영화 평론가 2인 × 이정재 | 월 2회 | 이번 달 FFWD 인기작 3편을 평론가가 시네마틱 렌즈로 분석 |
| **시네마 디스커버리** | 영상 숏폼(15분) | 졸업생 감독 3인 라운드테이블 | 월 1회 | 같은 주제(빛, 소리, 침묵, 색)로 각자의 접근법을 나누는 라운드테이블 |

### "감독의 밤" 에피소드 구조 (예시)

```
[00:00] 인트로 — FFWD Motion Signature (0.5초 필름 게이트 오프닝)
[00:30] 이정재: "오늘의 작품은 OOO 감독의 'OOO'입니다"
[01:00] 졸업 작품 키 장면 3분 재생 (FFWD 에피소드 1개 전편)
[04:00] "이 장면을 왜 이렇게 찍었나?" — 감독이 프레임 단위로 설명
[15:00] "만약 예산이 10배였다면?" — What-If 시나리오
[30:00] 이정재의 멘토링 피드백
[45:00] 관객 Q&A (FFWD Circle 멤버 우선)
[55:00] 다음 에피소드 예고 + 작품 FFWD 링크
[60:00] 아웃트로 — "Fast Forward Cinema."
```

### 배포 채널

| 채널 | 형태 | 접근성 |
|------|------|--------|
| **FFWD 앱/웹** | 풀 영상 + 오디오 | Circle 멤버 우선 공개 (1주 선공개) |
| **YouTube** | 풀 영상 | 1주 후 공개 |
| **Apple Podcasts / Spotify** | 오디오 | 동시 공개 |
| **Instagram Reels / TikTok** | 하이라이트 클립(60초) | 즉시 공개 — 유입 퍼널 |

### 수익 모델

- **직접 수익**: 팟캐스트 내 프리미엄 스폰서십 (브랜드 콜라보)
- **간접 수익**: 작품 조회수 증가 → 유료 열람 전환
- **멤버십 연계**: Circle 멤버 우선 접근 → 멤버십 가입 동기

### 벤치마크 vs 차별화

| A24 팟캐스트 | FFWD "프레임 너머" |
|-------------|----------------------|
| 기성 A-list 배우 대담 (Joaquin Phoenix, Michelle Yeoh) | **거장 × 신인**의 세대 연결 — 업계 유일 |
| 영화 개봉 시즌에 맞춘 홍보성 | **교육 + 멘토링** 성격 — 졸업생 성장 스토리 |
| 오디오 중심 | **영상 + 작품 재생**이 통합된 멀티미디어 |
| 팬 참여 없음 | **Circle 멤버 Q&A** 양방향 참여 |

## 2. FFWD Circle 멤버십

> A24의 AAA24 멤버십($9.99~$99)에서 영감. 그러나 FFWD Circle은 **"관객이 곧 후원자"**라는 철학을 담는다.

### 콘셉트

**"FFWD Circle"** — 원 안에 모인 사람들. 영화를 사랑하는 사람들이 차세대 영화인을 직접 후원하고, 그 성장을 함께 지켜보는 내부자 커뮤니티.

> "당신의 멤버십은 당신의 취향을 사는 것이 아니라, 다음 봉준호의 첫 프레임에 투자하는 것입니다."

## 3. 숏폼 연계 굿즈 — "프레임 속 오브젝트"

> NEON의 "영화=브랜드" 전략 + A24의 세계관 확장 머천다이즈($850 탁구대)에서 영감. FFWD는 **졸업 작품 속 소품과 세계관**을 물리적 오브젝트로 확장한다.

### 콘셉트

**"Frame Objects"** — 프레임 속에 있던 것이 프레임 밖으로 나온다.

졸업 작품 속 핵심 소품, 의상, 포스터, 대사 등을 한정판 굿즈로 제작. 작품의 세계관이 관객의 일상으로 확장된다.

### 굿즈 카테고리

| 카테고리 | 예시 | 가격대 | 생산 방식 |
|----------|------|--------|----------|
| **아트 프린트** | 작품 키 프레임을 미술 프린트로 제작 (A3/A2, 번호 에디션) | ₩25,000~₩50,000 | 주문 제작 (POD) |
| **대사 티셔츠** | 작품 속 인상적인 대사를 타이포그래피로 | ₩35,000~₩45,000 | 분기별 한정판 |
| **감독 노트 Zine** | 감독의 콘티, 스케치, 메모를 엮은 미니 진(Zine) | ₩15,000~₩25,000 | 분기별 발행 |
| **작품 소품 레플리카** | 작품 속 핵심 소품의 미니어처 또는 레플리카 | ₩20,000~₩80,000 | 인기작 한정 |
| **OST Vinyl / CD** | 작품 OST를 피지컬 미디어로 | ₩30,000~₩45,000 | 시즌별 컴필레이션 |
| **포스터** | @filmoment 스타일의 시네마틱 스틸 포스터 | ₩15,000~₩30,000 | 상시 |

### 플랫폼 내 연동 UX

```
[에피소드 시청 화면]
├── 3분 에피소드 재생
├── 에피소드 종료 후 →
│   ├── "이 프레임을 소장하세요" — 키 프레임 아트 프린트 링크
│   ├── "감독의 노트를 읽어보세요" — Zine 링크
│   └── "이 작품의 OST" — 음원/Vinyl 링크
└── 작품 상세 페이지 하단 →
    └── "Frame Objects" 섹션 — 해당 작품 관련 전체 굿즈
```

> A24 방식: 영화 상세 페이지 안에 해당 영화 머천다이즈 직접 노출 — FFWD도 동일 패턴 적용

### 운영 전략

**Phase 1 (Year 1)**: 아트 프린트 + 포스터만 — POD 방식으로 재고 리스크 없이 시작
**Phase 2 (Year 2)**: 대사 티셔츠 + Zine 추가 — 분기별 한정판으로 FOMO 조성
**Phase 3 (Year 3)**: 소품 레플리카 + Vinyl — 콜렉터 문화 본격 조성

### "Sold Out" 아카이브 전략 (A24 차용)

- 품절 상품을 삭제하지 않고 **"Sold Out" 표시로 유지**
- "이 작품의 굿즈를 놓치셨나요? Circle 멤버가 되면 다음 드롭을 먼저 만납니다"
- 희소성/FOMO → Circle 멤버십 전환 동기

## 4. OST 연계 — "FFWD Sound"

> A24 Music(자체 음악 레이블 + OST Vinyl 유통)에서 영감. FFWD는 **3분짜리 숏폼의 음악**이라는 독특한 포맷을 활용한다.

### 콘셉트

**"FFWD Sound"** — 3분짜리 영화에 3분짜리 음악. 숏폼 콘텐츠의 OST를 독립 음원으로 발매하는 파이프라인.

### 왜 FFWD OST인가?

- 졸업 작품의 음악감독(작곡가)도 대부분 음대 졸업생 — **"차세대 영화 음악가"**의 발굴
- 3분 러닝타임 = 음원 1곡 길이와 동일 — 별도 편집 없이 바로 음원화 가능
- 숏폼 특성상 음악이 영상의 50% 이상 차지 — OST의 존재감이 장편보다 크다

### OST 파이프라인

```
[졸업 작품 등록] 
  → 음악 트랙 메타데이터 분리 (작곡가, 곡명, 장르)
  → [자동] Whisper로 음악 구간 식별 + 분리
  → [큐레이션] 음악 큐레이터가 OST 품질 검증
  → [발매]
     ├── 디지털: Spotify, Apple Music, YouTube Music
     ├── FFWD 앱 내 음원 스트리밍
     └── Vinyl/CD: 시즌 컴필레이션 (분기 1회)
```

### 프로그램 라인업

| 프로그램 | 설명 | 주기 |
|----------|------|------|
| **Weekly Sound** | 이번 주 베스트 작품 3편의 OST를 플레이리스트로 | 매주 |
| **FFWD OST Vol. N** | 분기별 베스트 OST 10~15곡 컴필레이션 앨범 | 분기별 |
| **Score Spotlight** | 작곡가 인터뷰 + 작곡 과정 비하인드 | 월 1회 |
| **Vinyl Drop** | 연간 베스트 OST Vinyl 한정판 (번호 에디션) | 연 1회 |

### 수익 모델

| 수익원 | 예상 단가 | 비고 |
|--------|----------|------|
| 디지털 스트리밍 | ₩7~₩10/재생 | Spotify/Apple Music 배분 |
| FFWD 앱 내 청취 | 구독료에 포함 | Circle 멤버 혜택 |
| Vinyl 한정판 | ₩40,000~₩55,000 | 연 1회, 500장 한정 |
| 컴필레이션 CD | ₩18,000~₩25,000 | 분기별 |
| 음원 라이센싱 | ₩50만~₩500만/건 | 광고, 다른 영상 콘텐츠에 사용 |

### 작곡가 수익 배분

- **창작자 70% 원칙 동일 적용** — 음악도 영상과 같은 기준
- 디지털 스트리밍 수익의 70%를 작곡가에게
- Vinyl/CD 판매 수익의 70%를 작곡가에게
- "음악감독도 FFWD 크리에이터" — 영상 감독과 동등한 대우

## 5. 아트 디렉터 2차 연계 — "FFWD Visual Lab"

> @filmoment의 시네마틱 스틸 큐레이션에서 영감. FFWD는 단순히 스틸컷을 뽑는 것이 아니라, **전문 아트 디렉터가 작품의 비주얼 세계관을 2차 창작**한다.

### 콘셉트

**"FFWD Visual Lab"** — 졸업 작품의 프레임을 예술 작품으로 승격시키는 비주얼 아티스트 프로그램.

두 가지 역할:

1. **썸네일 아트 디렉터**: 각 에피소드의 키 비주얼(썸네일)을 @filmoment 수준의 미학으로 큐레이션
2. **포스트 아티스트 디렉터**: 작품의 세계관을 2차 비주얼(포스터, 아트 프린트, 모션 그래픽)로 확장

### 1) 썸네일 아트 디렉터

> "썸네일 하나가 클릭을 결정한다. 그러나 FFWD의 썸네일은 클릭을 넘어 예술이다."

| 역할 | 설명 |
|------|------|
| **키 프레임 선정** | 3분 에피소드에서 가장 강렬한 1프레임을 선택 — 감정, 구도, 빛의 3요소 기준 |
| **컬러 그레이딩** | @filmoment 스타일의 따뜻한 필름 톤으로 통일 (FFWD Colorgraph 적용) |
| **타이포 오버레이** | NEON 스타일의 비평 인용 또는 감독 코멘트를 시네마틱하게 오버레이 |
| **시리즈 통일감** | 같은 감독의 작품들이 피드에서 하나의 "전시"처럼 보이도록 비주얼 시스템 유지 |

**운영 구조**:
- **인하우스 아트 디렉터 1명** (Year 1) → 2~3명 (Year 3)
- **프리랜서 네트워크**: 대학 시각디자인과 졸업생을 아트 디렉터 풀로 구축
- **AI 보조**: 키 프레임 후보 3~5장을 AI가 자동 추출 → 아트 디렉터가 최종 선택

### 2) 포스트 아티스트 디렉터 — 2차 비주얼 확장

> "졸업 작품은 3분이지만, 그 세계관은 무한하다."

| 2차 확장물 | 설명 | 용도 |
|-----------|------|------|
| **시네마틱 포스터** | 영화 포스터 수준의 아트워크 제작 | SNS, 굿즈(프린트), 영화제 출품용 |
| **모션 포스터** | 키 프레임에 미세한 움직임(시네마그래프) 추가 | Instagram/TikTok 홍보, 앱 내 배너 |
| **캐릭터 아트** | 작품 속 캐릭터를 일러스트/아트워크로 재해석 | 굿즈(스티커, 엽서), 팬아트 문화 조성 |
| **타이틀 시퀀스** | 각 시리즈의 오프닝 타이틀 시퀀스(10초) 제작 | 앱 내 재생 시 브랜드 경험 강화 |
| **영화제 프레스킷** | Siena Film 스타일의 케이스 스터디 비주얼 | FFWD Awards, 외부 영화제 출품 |

### 아트 디렉터 ↔ 크리에이터 협업 모델

```
[졸업생 감독]
  ↓ 작품 등록
[AI 자동 처리]
  ↓ 키 프레임 후보 5장 추출 + 자동 컬러 분석
[썸네일 아트 디렉터]
  ↓ 최종 키 프레임 선택 + 컬러 그레이딩 + 타이포
[감독 승인]
  ↓ 감독이 최종 썸네일 확인/피드백
[게시]
  ↓ 에피소드 썸네일 + SNS용 비주얼 동시 배포

[인기작 선정 시]
  ↓
[포스트 아티스트 디렉터]
  ↓ 시네마틱 포스터 + 모션 포스터 + 캐릭터 아트
[굿즈 라인 론칭]
  ↓ Frame Objects 스토어에 아트 프린트/포스터 등록
```

### 수익 모델 & 크리에이터 경제

- **아트 디렉터 수익**: 아트 프린트/포스터 판매 수익의 **30%** (작품 감독 40%, 플랫폼 30%)
- **포스트 아티스트 수익**: 2차 비주얼 굿즈 판매 수익의 **20%** (감독 40%, 아티스트 20%, 플랫폼 40%)
- **커리어 트랙**: "FFWD Visual Lab 출신"이 영화 포스터/타이틀 시퀀스 업계에서 브랜드가 되도록

### 자기 강화 루프 (Collins Self-Reinforcing Loop)

1. **작품 등록** → Visual Lab이 시네마틱 썸네일 제작 → SNS 확산
2. **SNS에서 유입** → 3화 무료 시청 → Circle 멤버십 전환
3. **Circle 멤버** → 팟캐스트 Q&A 참여 + 투표 가중치 → 커뮤니티 활성화
4. **인기작 탄생** → Frame Objects 굿즈 + OST 발매 → 추가 수익
5. **수익** → 차세대 기금(30% 환원) → 더 많은 졸업 작품 지원
6. **더 많은 작품** → Visual Lab의 더 다양한 비주얼 → 더 강한 SNS 확산

> **이 루프가 한 번 돌아가면, FFWD는 단순 스트리밍 플랫폼이 아니라 차세대 영화인의 생태계가 된다.**
