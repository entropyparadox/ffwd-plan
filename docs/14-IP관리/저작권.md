# IP & 저작권 관리 프레임워크

> 졸업작품의 IP 소유권은 생각보다 복잡하다. 법적 기반 없이 플랫폼을 운영하면 성장할수록 리스크가 커진다.

## 졸업작품 IP 소유권 현황

### 문제 정의

대학 졸업작품의 IP 소유권은 **학교마다 다르다**:

| 유형 | 비율 (추정) | 설명 |
|------|-----------|------|
| 학생 100% 소유 | ~40% | 대부분의 사립대, 일부 국립대 |
| 학교-학생 공동 소유 | ~30% | 학교 장비/시설 사용 조건 |
| 학교 소유 (학생 사용권) | ~20% | 일부 국립대, 국비 지원 프로젝트 |
| 불명확 | ~10% | 계약서 없이 진행된 경우 |

### FFWD 영향

- **학교 소유/공동 소유** 작품은 학교의 동의 없이 상업적 이용 불가
- **불명확** 사례는 분쟁 소지 — 플랫폼 성장 시 폭탄
- IP 라이선싱(장편 전환)은 소유권이 명확해야 진행 가능

## 필수 법적 프레임워크

### 1. 업로드 이용 약관 (Terms of Service)

```
[창작자 → FFWD 업로드 시 동의 사항]

1. 권리 보증: "본인이 이 작품의 저작권자이거나, 
   저작권자의 이용 허락을 받았음을 확인합니다."

2. 이용 허락: "FFWD 플랫폼에서의 스트리밍, 
   자막 번역, 홍보 목적 사용을 허락합니다."

3. 비독점: "이 작품을 다른 플랫폼에도 게재할 수 있습니다."
   (독점 계약은 COMMISSION 프로그램에서만)

4. 수익 배분: "유료 열람/구독 수익의 70%를 
   창작자에게 배분합니다."

5. 제3자 권리: "작품에 등장하는 모든 인물의 
   초상권 동의를 확보했음을 확인합니다."
```

### 2. 대학 MOU 표준 계약

| 조항 | 내용 |
|------|------|
| IP 소유권 확인 | 학교가 학생에게 IP를 양도/이용허락하는 구조 확인 |
| 포괄 동의 | 졸업작품 FFWD 게재에 대한 학교 차원 동의 |
| 학생 개별 동의 | 학교 포괄 동의와 별개로 학생 개인 동의 필수 |
| 수익 배분 | 학교 공동 소유 시: 학교 10% + 학생 60% + 플랫폼 30% |
| 독점 여부 | 비독점 기본, 독점은 별도 계약 |

### 3. COMMISSION (투자 제작) 계약

| 조항 | 내용 |
|------|------|
| 제작비 지원 | 편당 최대 5,000만 원 |
| IP 소유 | FFWD 50% + 창작자 50% (공동 소유) |
| 독점 기간 | FFWD 독점 스트리밍 2년 |
| 장편 전환 | 공동 소유로 장편 제작 시 양측 합의 필요 |
| 2차 저작물 | 수익 50:50 배분 |

## IP 라이선싱 구조 (장편 전환)

```
[졸업작품 (3분 숏폼)]
    ↓ FFWD에서 화제
    ↓ 조회수 + 전문가 평가 상위
[IP 라이선싱 협상]
    ├── OTT 독점 → 작품당 ₩500만~₩5,000만
    ├── 장편 영화화 → 원작료 + 공동 투자
    └── 해외 리메이크 → 국가당 라이선스 피
```

### 장편 전환 수익 배분

| 당사자 | 비율 | 근거 |
|--------|------|------|
| 원작 창작자 | 30~40% | 원작 + 참여 인센티브 |
| FFWD | 20~30% | 발굴 + 플랫폼 기여 |
| 제작사 | 30~40% | 장편 제작 투자 |
| 학교 (해당 시) | 5~10% | 공동 소유 시 |
