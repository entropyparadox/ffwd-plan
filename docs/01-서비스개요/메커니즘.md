# 핵심 메커니즘

> "Build the Next Hottest Thing."

## 자기 강화 루프

가장 강력한 플랫폼은 자기 강화 루프를 갖는다. 콘텐츠가 창작자를 끌어오고, 창작자가 관객을 끌어오고, 관객이 수익을 만들고, 수익이 더 많은 콘텐츠를 만든다. 이 루프가 한 번 돌아가면, 경쟁자는 단일 요소가 아니라 시스템 전체를 복제해야 한다.

## 서비스 플로우

```
[졸업작품 등록] → [3분 단위 에피소드 분할] → [3화 무료 / 나머지 유료]
        ↓
[전문가 투표·평가] + [대중 참여(조회·좋아요·열람·투표)]
        ↓
[연간 통합 랭킹] → [시상식/영화제 개최]
        ↓
[상위 30팀 × 3억 제작지원비] → [신작 30편 제작·등록]
        ↓
[자동 자막] → [글로벌 서비스] → [100개국+ 확장]
```

## 핵심 루프: 콘텐츠 → 발굴 → 지원 → 콘텐츠

FFWD의 핵심은 **자기 강화 루프(self-reinforcing loop)**이다:

1. 대학에서 졸업작품 등록 → 콘텐츠 풀 확보
2. 전문가 + 대중이 평가 → 우수 작품 발굴
3. 시상식에서 제작지원비 제공 → 창작자 동기 부여
4. 지원받은 30팀이 신작 제작 → 더 높은 수준의 콘텐츠
5. 신작이 플랫폼에 등록 → 관객 유입 증가
6. 관객 증가 → 수익 증가 → 더 많은 지원 가능

이 루프가 한 번 돌아가면, 매년 콘텐츠 품질과 관객 규모가 동시에 성장한다.

### 6대 임팩트 축 맵핑

이 자기 강화 루프는 FFWD의 6대 임팩트 축 중 4가지를 동시에 달성한다:

- **시장 차별성 확립** — 졸업작품 스트리밍이라는 새 카테고리 창출
- **문화적 관련성 창출** — 영화제 + 시상식으로 문화적 권위 획득
- **성장 가속** — 루프가 돌 때마다 기하급수적 성장
- **시장 기회 개척** — 글로벌 자막으로 100개국+ 진출

## 웹툰 모델 적용

네이버웹툰이 "만화"를 "웹툰"으로 재정의한 것처럼, FFWD는 "졸업작품"을 "시네마틱 숏폼"으로 재정의한다. 기존 카테고리에서 경쟁하지 않고, 새 카테고리의 룰을 쓴다.

| 웹툰 (네이버/카카오) | FFWD | 전략적 의미 |
|---|---|---|
| 1화 단위 연재 | 3분 단위 에피소드 | 몰입형 네이밍: "에피소드" |
| 첫 3화 무료 | 첫 3화 (9분) 무료 | 선택 설계 (무료 체험 → 유료 전환) |
| 나머지 유료 (200~300원) | 나머지 유료 (300~500원) | 프리미엄 가격 정당화 |
| 쿠키/코인 충전 | 코인 충전 모델 | 즉각적 차별화 |
| 작가 수익 배분 70% | 창작자 수익 배분 70% | Creator First 원칙 |
| 주간 인기 랭킹 | 전문가+대중 통합 랭킹 | 검증 가능한 품질 |
| 웹툰 → 드라마/영화화 | 졸업작품 → 장편 제작지원 | 가치-시장 적합성 |
