# 7대 수익 모델

> "Expand Your Value. Command Your Market."

## 6대 임팩트 축 맵핑

| 수익 모델 | 임팩트 축 |
|---|---|
| 콘텐츠 열람 수수료 | 프리미엄 가격 정당화 — 시네마틱 품질에 프리미엄 과금 |
| 프리미엄 구독 | 시장 차별성 확립 — 구독자만의 독점 경험 |
| 광고 수익 | 문화적 관련성 창출 — 브랜드가 문화 후원자가 됨 |
| 영화제/이벤트 | 문화적 관련성 창출 — 연간 문화 이벤트로 자리매김 |
| IP 라이선싱 | 포트폴리오 최적화 — 콘텐츠 자산의 다채널 수익화 |
| 국가지원금 | 시장 기회 개척 — 공공 파트너십으로 성장 가속 |
| 글로벌 로열티 | 성장 가속 — 국가별 프랜차이즈 확장 |

## 수익 구조 개요

```
[매출 구조]
├── 1. 콘텐츠 열람 수수료 (B2C) ← 핵심
├── 2. 프리미엄 구독 (B2C)
├── 3. 광고 수익 (B2B)
├── 4. 영화제/이벤트 수익 (B2B2C)
├── 5. IP 라이선싱 (B2B)
├── 6. 국가지원금 (G2B)
└── 7. 글로벌 로열티 (B2B)
```

### ① 콘텐츠 열람 수수료 (핵심 — 웹툰 모델)

> "Press Play on the Future." — 3분짜리 프레임에도 가치가 있다.

| 항목 | 내용 |
|---|---|
| 모델 | 에피소드별 결제 (웹툰 방식) |
| 가격 | 에피소드당 300~500원 |
| 무료 분량 | 첫 3화 (약 9분) 무료 |
| 수수료 배분 | 플랫폼 30% : 창작자 70% |

### ② 프리미엄 구독 (월정액)

> "Your Vision. Your Frame." — 당신만의 시네마틱 경험을 소유하라.

| 항목 | 내용 |
|---|---|
| 가격 | ₩4,900/월 또는 ₩49,000/년 |
| 혜택 | 전 콘텐츠 무제한 + 전문가 분석 리포트 + 독점 비하인드 + 영화제 우선 투표권 |

### ③ 광고 수익

| 항목 | 내용 |
|---|---|
| 모델 | 무료 콘텐츠(3화) 앞 프리롤 광고 + 배너 |
| CPM | ₩3,000~₩8,000 (타겟팅 기반) |
| 브랜드 스폰서십 | 특정 작품/시상식 네이밍 스폰서 |

### ④ 영화제/이벤트 수익

| 항목 | 내용 |
|---|---|
| 티켓 | 현장 관람 ₩30,000~₩50,000 |
| F&B/브랜드 부스 | 부스당 ₩500만~₩2,000만 |
| 스폰서십 | 메인 스폰서 ₩5억+ |
| 지자체 보조금 | 행사당 ₩1억~₩3억 |

### ⑤ IP 라이선싱

우수작 OTT 독점 라이선스 (넷플릭스, 왓챠, 웨이브 등) — 작품당 ₩500만~₩5,000만

### ⑥ 국가지원금

| 항목 | 내용 |
|---|---|
| 콘텐츠진흥원 | 90억원 (3년 분할) |
| 지자체 | 연간 ₩3억~₩5억 |
| 문체부/산자부/중기부 | 추가 지원 프로그램 |

### ⑦ 글로벌 로열티 (Year 3+)

> "Enter New Markets with an Edge."

국가별 파트너와 라이선스/프랜차이즈 — 국가당 연간 ₩1억~₩5억

::: tip MVP 우선순위 참고
MVP에서는 **프리미엄 구독 + 광고**를 우선 도입합니다. 에피소드별 과금(웹툰 모델)은 Phase 2에서 사용자 행동 데이터를 확보한 후 검증하여 도입할 예정입니다. 초기에 두 가지 수익 모델을 동시 운영하여 어떤 모델이 유료 전환율이 높은지 A/B 테스트를 진행합니다.
:::
