# 리스크 분석 & 대응 전략

> "겉보기에 합의처럼 보이는 것이 실은 무리 행동이 성숙함으로 위장한 것일 수 있다." — Dario Amodei

## 🔴 핵심 리스크 5가지

### 1. Quibi 리스크 — "유료 프리미엄 숏폼" 시장이 검증되지 않았다

| 항목 | Quibi | FFWD |
|------|-------|-----------|
| 투자액 | $17.5억 (₩2.3조) | ₩5억 (초기 자본) |
| 콘텐츠 | 할리우드 스타 기반 | 대학 졸업작품 기반 |
| 가격 | $4.99~$7.99/월 | ₩4,900/월 |
| 최대 구독자 | ~50만 (유료) | — |
| 결과 | **6개월 만에 폐업** | — |

**Quibi 실패 핵심 원인:**
1. **PMF(Product-Market Fit) 미검증 상태에서 대규모 투자** — 수십억 달러를 투입하기 전에 사용자가 유료 숏폼을 원하는지 확인하지 않음
2. **무료와 경쟁** — TikTok/YouTube/Instagram이 무료인 상황에서 "프리미엄 숏폼" 구매자가 없었음
3. **모바일 전용, 공유 불가** — 바이럴리티 제로
4. **콘텐츠가 "must-watch"가 아니었음** — 셀러브리티 이름 ≠ 시청 습관
5. **코로나로 "이동 중 시청" 유스케이스 소멸**

**FFWD 대응 전략:**
- MVP에서 **유료 과금 검증을 최우선**으로 설정 — 전체 구축 전에 소규모 테스트
- 웹+모바일+공유 기능 Day 1 — Quibi의 모바일 전용 실수 회피
- 에피소드별 과금 대신 **프리미엄 구독 + 무료(광고) 하이브리드**로 시작
- 이정재 브랜드를 **분산된 셀러브리티 전략이 아닌 단일 큐레이터**로 집중

### 2. TAM 천장 — 콘텐츠 공급원이 너무 좁다

| 데이터 | 수치 |
|--------|------|
| 한국 연극영화과 대학 | ~200개교 |
| 연간 졸업생 | ~1,500~2,500명 |
| 졸업작품 연간 생산량 | ~2,000~4,000편 |
| "시네마틱 퀄리티" 비율 | ~10~20% |
| **실질 연간 콘텐츠 풀** | **~400~800편** |

연간 400~800편은 주간 8~15편 수준 — 매일 새 콘텐츠를 기대하는 관객을 유지하기엔 부족합니다.

**대응 전략:**
1. **소싱 확대**: 졸업작품 외에도 독립영화인, 1인 감독, 영화제 미선정작 수용
2. **글로벌 Day 1**: Year 1부터 일본(자주영화), 대만, 동남아 영화학교와 MOU
3. **리메이크/시리즈**: 우수작의 시리즈화, 감독 후속작 위탁 제작
4. **마스터클래스 콘텐츠**: 교육 콘텐츠로 비소설 영역 확장

- BEP 시점을 Year 2 Q3 → **Year 3~4**로 현실화
- 자세한 분석은 [매출 전망 (현실화)](/12-단위경제/매출현실화) 참조

### 4. 국가지원금 의존 구조

기획서 원안은 매년 ₩30억(3년간 ₩90억)을 콘텐츠진흥원에서 수령하는 것을 가정합니다.

**현실 검증:**
- 콘텐츠진흥원 단일 프로젝트 지원 규모: 보통 ₩1~5억/년
- 신생 JV에 연간 ₩30억은 **매우 어려운 시나리오**
- 지원금은 심사 기반이며, 매년 재신청 필요

**대응 전략:**
1. 지원금 규모를 **₩3~5억/년**으로 현실화
2. 제작지원비(30팀×3억=₩90억)는 **스폰서십 + 자체수익**으로 충당
3. 영화제 스폰서십(₩5억+)이 진짜 자금원
4. 다수 지원 프로그램 병행 지원 (문체부, 산자부, 중기부, 지자체)

---

### 5. 이정재 단일 의존 리스크

기획서 전체가 "이정재가 추천하면 본다"에 의존합니다.

**리스크:**
- 이정재 스케줄/건강/이미지 리스크
- 개인 브랜드 ≠ 플랫폼 브랜드 (이정재가 떠나면?)
- Quibi도 할리우드 셀러브리티에 의존했으나 실패

**대응 전략:**
1. 이정재를 **초기 점화(ignition)**로 활용, 시스템이 자체 작동하는 구조로 전환
2. Year 2부터 "이정재 없이도 FFWD가 작동하는가?" 테스트
3. 전문가 투표 시스템 + 커뮤니티 큐레이션이 **이정재를 대체 가능한 수준**까지 성장
4. 복수의 "Founding Curator" 추가 (해외 감독 등)
