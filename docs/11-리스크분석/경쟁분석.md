# 경쟁 환경 분석

> "Don't Become the Best. Become the Only."

## 경쟁 포지셔닝 맵

```
                  높은 영화적 품질
                       ↑
                       |
          FFWD ● |  ● MUBI (장편)
                       |
   긴 형식 ←───────────┼───────────→ 짧은 형식
                       |
             ● Netflix |   ● TikTok
                       |   ● YouTube Shorts
                       |   ● Instagram Reels
                       ↓
                  낮은 영화적 품질
```

**FFWD의 포지션: "짧은 형식 + 높은 영화적 품질"** — 이 사분면에는 현재 아무도 없다.

## 직접 경쟁사 상세 분석

### Mometu (2026년 3월 론칭) ⚠️ 직접 경쟁자

| 항목 | Mometu | FFWD |
|------|--------|-----------|
| 론칭 | 2026년 3월 | 계획 중 |
| 모델 | 단편영화 경쟁 플랫폼 | 시네마틱 숏폼 스트리밍 |
| 크리에이터 수익 | **100% 환원** | 70% 배분 |
| 차별점 | 경쟁/대회 중심 | 큐레이션 + 스트리밍 + 영화제 |
| 브랜드 파워 | 없음 | 이정재 |

**대응:** Mometu는 "경쟁/대회" 모델이고, FFWD는 "큐레이션/스트리밍" 모델. 직접 충돌보다는 **보완적 포지셔닝** 가능. 단, 100% 크리에이터 환원은 마케팅적으로 강력 — FFWD의 70% 배분에 대한 명확한 가치 정당화 필요.

### MUBI — 니치 스트리밍 성공 사례

| 항목 | 수치 |
|------|------|
| 매출 (2024) | **$77M** |
| 고객 수 | 840만 |
| 밸류에이션 (2026) | **$1.1B (유니콘)** |
| 총 펀딩 | $135.65M |
| 가격 | ~$5.99~$13.99/월 |
| 모델 | 큐레이션 30편 회전 라이브러리 |
| 직원 | ~400명 |

**MUBI에서 배울 것:**
1. 니치 스트리밍도 유니콘 가능 — **큐레이션이 차별점**
2. "Less is More" — 무한 카탈로그 대신 엄선된 소수
3. 영화 배급/제작으로 수직 확장 (MUBI도 극장 배급 진출)
4. **10년의 인내** — 하루아침에 되지 않음

### Quibi — $17.5억 실패 케이스

| 항목 | 수치 |
|------|------|
| 투자액 | **$17.5억** |
| 결과 | **6개월 만에 폐업** (2020.04~10) |
| 최대 유료 구독자 | ~50만 |
| 에피소드 길이 | 7~10분 |
| 가격 | $4.99~$7.99/월 |

**Quibi 교훈 → FFWD 적용:**
1. PMF 검증 전 대규모 투자 금지 → **린 MVP로 시작**
2. 무료와 경쟁 → **무료 티어(광고) 필수 제공**
3. 공유 불가 → **SNS 공유, 인트로 클립 자동 생성 Day 1**
4. 셀러브리티 분산 → **이정재 단일 큐레이터로 집중**

## FFWD 방어벽 (Moat) 분석

| 방어벽 | 강도 | 설명 |
|--------|------|------|
| 이정재 브랜드 | ⭐⭐⭐⭐ | 복제 불가능한 개인 브랜드. 단, 단일 의존 리스크 |
| 큐레이션 네트워크 | ⭐⭐⭐ | 전문가 투표단 구축에 시간 필요. 선점 효과 |
| 대학 파트너십 | ⭐⭐⭐ | 200개교 MOU는 후발주자가 복제하기 어려움 |
| 카테고리 정의자 | ⭐⭐⭐⭐ | "시네마틱 숏폼 = FFWD" 등식 선점 |
| 기술 스택 | ⭐⭐ | 자막 파이프라인은 범용 기술. 기술 자체는 방어벽 아님 |
| 콘텐츠 독점 | ⭐⭐ | 졸업작품은 독점 계약 어려움. 다중 플랫폼 게재 가능 |

> **결론**: FFWD의 진짜 방어벽은 기술이 아니라 **"브랜드 + 네트워크 + 카테고리 정의"**의 결합이다. 이 세 가지가 동시에 작동할 때 경쟁자는 단일 요소가 아니라 시스템 전체를 복제해야 한다.
