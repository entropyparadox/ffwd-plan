## Month 3~4: 클로즈 베타

### 클로즈 베타 목표

- **작품 200편+ 등록** (20개교 × 10편)
- **베타 사용자 500명** (대학생 + 영화인)
- **유료 과금 첫 검증** — 에피소드 구매가 실제로 발생하는지

### 클로즈 베타 검증 항목

| 가설 | 측정 지표 | 성공 기준 | 실패 시 대응 |
|------|----------|----------|------------|
| 졸업작품에 유료 결제 의향 있음 | 유료 전환율 | 3%+ | 무료(광고) 모델로 피봇 |
| 3분 에피소드 분할이 시청 완주율 높임 | 완주율 | 70%+ | 분할 단위 조정 (5분/10분) |
| 자동 자막 품질이 시청에 방해 안 됨 | 자막 관련 불만 비율 | <10% | QA 인력 강화 |
| 이정재 추천이 CTR 높임 | 이정재 추천 vs 일반 CTR | 2배+ | 추천 포맷 변경 |

### 콘텐츠 소싱 (Day 1 라이브러리 확보)

| 소스 | 편수 | 방식 |
|------|------|------|
| MOU 체결 대학 졸업작품 | 200편 | 학교별 10편 선별 |
| 이정재 네트워크 (감독 추천작) | 30편 | 직접 큐레이션 |
| 영화제 미선정작 (독립영화인) | 50편 | 공모 |
| 마스터클래스 (이정재 + 게스트) | 10편 | 자체 제작 |
| **합계** | **~290편** | |
