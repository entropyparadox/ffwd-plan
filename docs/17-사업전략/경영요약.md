# 경영 요약 (Executive Summary)

> 작성일: 2026-04-05 · CEO Jarvis (키넥션 JV 사업 분석) · 소스: ffwd-plan.vercel.app 기획서 전체 + 시장 리서치

---

**FFWD**는 "시네마틱 숏폼 스트리밍 플랫폼"으로, 전국 200여 개 연극영화과 졸업작품을 3분 단위 에피소드로 서비스하고, 전문가 투표 + 연간 시상식 + 제작지원비를 통해 차세대 영화인을 발굴·육성하는 생태계입니다.

## 핵심 베팅 (Counter-Consensus)

- **"졸업작품에 상업적 가치가 있다"** — 매년 2,000~4,000편이 어둠 속에 묻히지만, 봉준호·박찬욱도 졸업작품에서 시작
- **"숏폼에서도 프리미엄 과금이 된다"** — Quibi가 실패한 영역이지만, 웹툰 모델(에피소드별 과금)로 접근하면 가능
- **"시네마틱 숏폼"이라는 새 카테고리 창조** — 짧은 형식 + 높은 영화적 품질의 사분면에는 현재 아무도 없음
