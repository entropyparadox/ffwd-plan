# 5년 사업 설계

## Phase 1: ORIGIN — 카테고리 정의 (Year 1, 2027)

> "Build the Next Hottest Thing." — 새로운 카테고리를 만들고, 시장에 존재를 선언하는 해

### 목표

- **"대학 졸업작품을 3분 단위로 볼 수 있는 유일한 플랫폼"** 포지션 선점
- PMF(Product-Market Fit) 검증 — 유료 과금이 되는지 확인
- 이정재 브랜드를 레버리지한 초기 인지도 확보

### 핵심 마일스톤

| Q | 마일스톤 | 주요 작업 | KPI |
|---|---------|----------|-----|
| Q1 | JV 설립 + MVP 개발 | 법인 설립, 팀 10명, 기술 스택 확정, 50개교 MOU | 법인 등기 완료 |
| Q2 | 클로즈 베타 출시 | 웹 플레이어, 결제, 자동 자막, 500편 등록 | 작품 200편+ |
| Q3 | 오픈 베타 | 전문가 10인 투표 시작, SNS 마케팅 개시 | MAU 3,000 |
| Q4 | 정식 출시 | 프리미엄 구독, 유튜버 콜라보 론칭 | MAU 5,000, 유료 전환 3%+ |

### Year 1 재무 (보수적)

| 항목 | 금액 |
|------|------|
| 총 비용 | ₩8~10억 (인건비 ₩7억 + 인프라 ₩0.9억 + 마케팅 ₩2억) |
| 자체 매출 | ₩0.3~0.8억 |
| 국가지원금 | ₩3억 (복수 프로그램 병행) |
| 적자 | ₩-4~7억 (투자금으로 충당) |

### Year 1 핵심 검증 항목

1. **유료 과금 검증**: 에피소드당 ₩300~500에 결제하는 사용자가 존재하는가?
2. **이정재 효과 검증**: 이정재 추천 콘텐츠의 CTR/전환율이 일반 대비 유의미하게 높은가?
3. **대학 파트너십 확장성**: MOU 체결 → 실제 작품 등록 전환율은 어떤가?
4. **콘텐츠 품질 임계치**: "시네마틱 퀄리티" 졸업작품은 전체의 몇 %인가? (추정 10~20%)

## Phase 2: MOMENTUM — 시스템 깊이 확장 (Year 2, 2028)

> "Go From Growing Pains to High Performance." — 검증된 가설로 깊이를 만드는 해

### 목표

- 전문가 투표 시스템 + 랭킹으로 "FFWD가 추천하면 믿고 본다" 브랜드 신뢰 구축
- 제1회 FFWD Film Festival(FLF) 개최
- 수익 모델 다각화 (구독 + 광고 + 영화제)

### 핵심 마일스톤

| Q | 마일스톤 | 주요 작업 | KPI |
|---|---------|----------|-----|
| Q1 | 전문가 네트워크 | 평론가·유튜버 50인+ 확보, 랭킹 시스템 오픈 | 전문가 50인 |
| Q2 | 영화제 준비 | 지자체 유치 확정, 스폰서 2~3개 확보, 모바일 앱 출시 | 스폰서 ₩5억+ |
| Q3 | 🏆 제1회 FLF | 5일간 도시 분산형 영화제, 30팀 선정, 제작지원비 집행 | 관객 10,000명 |
| Q4 | MAU 30K 돌파 | 광고 시스템 도입, IP 라이선싱 1~2건 | MAU 30,000 |

### Year 2 재무 (보수적)

| 항목 | 금액 |
|------|------|
| 총 비용 | ₩15~20억 (인건비 ₩15억 + 인프라 ₩2.5억 + 마케팅 ₩5억 + 영화제 ₩3억) |
| 자체 매출 | ₩5~12억 |
| 국가지원금 + 지자체 | ₩4~7억 |

## Phase 5: PORTAL — 글로벌 포털화 (Year 5, 2031)

> "Don't Become the Best. Become the Only."

### 목표

- "시네마틱 엔터테인먼트 OS"로 포지션 전환
- 10개국+ 서비스, MAU 500K+
- FLF를 아시아 최대 숏폼 영화제로 성장 (30,000명+ 관객)
- 유니콘 경로 진입 (밸류에이션 ₩1,000억+)
- MUBI 벤치마크: 10년 걸려 $77M 매출, $1.1B 밸류에이션 달성 → FFWD는 5~7년 내 유사 경로 진입 목표

### Year 5 목표 숫자

| 지표 | 목표 |
|------|------|
| MAU | 300,000~500,000 |
| 연 자체 매출 | ₩80~150억 |
| 진출 국가 | 5~10 |
| FLF 관객 | 30,000명+ (멀티시티) |
| 글로벌 크리에이터 | 5,000명+ |
| 밸류에이션 | ₩1,000억~3,000억 |

### 장기 비전 (Year 5+)

- **"영화계의 GitHub"**: 전 세계 영화학교가 작품을 올리고, 글로벌 관객과 전문가가 발굴하는 생태계
- **AI 레이어 확장**: 좌뇌(분석)·우뇌(감성)·오른팔(실행) AI 시스템으로 참여형 포털 진화
- **100개국 서비스**: 자동 자막 + 문화 동기화로 국경 없는 시네마 플랫폼
