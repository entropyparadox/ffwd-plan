# FLF — FFWD Film Festival

> 영화제가 아니라, 시네마를 중심에 둔 글로벌 컬처 페스티벌.

## 컨셉: "CINECITY"

기존 영화제의 틀을 벗어나, 도시 전체가 무대가 되는 문화 축제로 재설계합니다.

### 디자인 공식

> Slush의 학생 주도 진정성 + Art Basel의 브랜드 아트 + SXSW의 컨버전스 + teamLab의 몰입 + Clermont-Ferrand의 숏필름 스케일 + ComplexCon의 문화적 희소성

### 기존 영화제 vs FLF

| 기존 영화제 | FLF "CINECITY" |
|---|---|
| 극장에서 영화 상영 | 도시 전체가 스크린 — 카페, 갤러리, 옥상, 광장 |
| 레드카펫 + 시상식 | 이머시브 설치 + 야간 프로그래밍 + 푸드 컬처 |
| 업계인 전용 | 크리에이터·관객·브랜드가 함께 만드는 축제 |
| 1주일 → 끝 | 365일 커뮤니티 + 연 1회 메인 이벤트 |
| 수동 관람 | "No Spectators" — 모든 참가자가 무언가를 만든다 |

## 페스티벌 개요

| 항목 | 내용 |
|---|---|
| 명칭 | FLF — FFWD Film Festival |
| 포지셔닝 | "아시아의 Sundance × Slush × Art Basel" |
| 기간 | 5일 + 위성 이벤트 2주 |
| 장소 | 도시 전체 분산형 (Fuorisalone 모델) |
| 규모 | 10,000명+ (Year 2) → 30,000명+ (Year 5) |
| 핵심 차별점 | 대학생이 만들고, 전문가가 발굴하고, 관객이 투자하는 구조 |

## 5일 프로그램

### 🌅 Day 1: ARRIVAL — 도시가 깨어나다

- Clubhouse 3개 개장 (Directors' House, Industry Hub, Audience Camp)
- Opening Immersive — XR/프로젝션 매핑 오프닝
- Night Cinema — 야외 상영 + DJ + 포장마차 빌리지

### 🎬 Day 2: DISCOVER — 발굴하다

- Capsule Screenings — 도시 곳곳 50석 소극장
- XR Village — VR/AR/AI 영화 전용 체험존
- Director's Table — 감독과 함께하는 12인 디너

### ⭐ Day 3: VOTE — 판단하다

- Expert Live Review — 전문가 심사 라이브 스트리밍
- Industry Marketplace — 숏필름 바이어 미팅
- 마스터클래스 — 이정재 감독 + 게스트 감독

### 🏆 Day 4: CELEBRATE — 축하하다

- Merch Lab — 한정판 굿즈 커스터마이징
- Cinema Supper — 수상작 테마 요리 팝업
- 🏆 시상식 — 30팀 선정 + 제작지원비 수여
- Closing Party — 라이브 공연 + 수상팀 축하

### 🌐 Day 5: CONNECT — 연결하다

- FLF Academy — 수상팀 멘토링 세션
- Global Network Mixer — 해외 파트너 기관 미팅
- Open Screen — 비선정작도 상영할 수 있는 열린 무대
- Closing Ceremony — 내년 개최지 + Artistic Director 발표

## 브랜드 파트너십 (Art Basel 모델)

> "로고가 아니라 작품을 만들어라." — 브랜드는 광고가 아니라 예술을 후원한다.

| 등급 | 투자 | 혜택 |
|---|---|---|
| Presenting Partner | ₩5억+ | Clubhouse 네이밍 + 이머시브 설치 커미션 |
| Experience Partner | ₩2억 | 테마 체험존 운영 + VIP 프로그래밍 |
| Culture Partner | ₩5,000만 | 한정판 콜라보 굿즈 + 브랜드 팝업 |
| F&B Partner | 수수료 기반 | Cinema Kitchen 내 팝업 레스토랑 운영 |
| Tech Partner | 현물 | XR Village 디바이스 제공 + 기술 쇼케이스 |
