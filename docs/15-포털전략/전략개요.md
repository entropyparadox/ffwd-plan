# 포털 빌드업 전략

> "콘텐츠로 매일 열게 만들고, AI로 참여하게 만들고, 생태계로 떠나지 못하게 만든다."

## 왜 포털인가?

FFWD가 "시네마틱 숏폼 스트리밍 플랫폼"으로만 남으면, 역사는 반복된다.

| 서비스 | 전략 | 결과 |
|--------|------|------|
| Quibi | 프리미엄 숏폼 + 셀러브리티 + 모바일 | **$1.75B 투자, 6개월 폐업** |
| ReelShort | 마이크로드라마 + 대량 제작 | 미국 모바일에서 넷플릭스 추월 (시청 시간) |
| TikTok | 참여형 숏폼 + 커머스 + 검색 포털 | 24개월 리텐션 70%+ |

단일 기능 스트리밍은 **콘텐츠 군비 경쟁**에 빠진다. 포털은 **생태계 경쟁**으로 전환한다.

## 포지셔닝: "시네마틱 엔터테인먼트 OS"

| ❌ 피하는 표현 | 이유 |
|--------------|------|
| "숏폼 스트리밍 플랫폼" | 넷플릭스/틱톡의 저가판으로 인식 |
| "엔터테인먼트 포털" | 네이버/카카오와 직접 경쟁 프레이밍 |

| ✅ 사용하는 표현 | 이유 |
|----------------|------|
| **"시네마틱 엔터테인먼트 OS"** | 카테고리 창조 — 시네마 × AI × 참여의 교차점 |

Dario Amodei의 인프라 레이어 사고를 그대로 적용한다:

> "Anthropic's strategy is to be what makes everything you open smarter. The layer underneath, not the app on top."
> — Dario Amodei, Anthropic CEO

FFWD는 시네마 세계 위에서 작동하는 **AI 인프라 레이어**다.

## 비전 문장

> **"FFWD는 시네마를 보는 곳이 아니라, 시네마 안에서 사는 곳이다."**

## 고유 포지션: "시네마틱 품질 × 높은 참여도"

```
                    콘텐츠 품질
                       ↑
                       |
         FFWD ●  |  ● Netflix
         (시네마틱 숏폼 |    (시네마틱 장편)
         + AI 참여)    |
                       |
    ───────────────────┼───────────────── 참여도 →
                       |
         ReelShort ●   |  ● YouTube
         (마이크로      |    (UGC + 크리에이터
          드라마)       |     생태계)
                       |
         TikTok ●      |
         (UGC 숏폼     |
         + 커머스)      |
```

- **넷플릭스**: 품질은 높지만 참여가 없다 (시청만)
- **TikTok**: 참여는 높지만 품질이 낮다 (UGC)
- **FFWD**: 둘 다 — AI가 가능하게 만든 새로운 카테고리

## Quibi에서 배우는 것

Quibi는 FFWD와 구조적으로 유사했다:

- 프리미엄 숏폼 ✓
- 셀러브리티 라인업 (스필버그, JLo, 케빈 하트) ✓
- 모바일 우선 ✓
- 결과: **$1.75B 투자, 6개월 폐업**

실패 핵심 원인:

> "그들은 사람들이 숏폼 콘텐츠의 제작 품질에 불만이 있다고 생각했다. 진짜 페인포인트는 '대화에 참여하지 못하는 것'이었다. TikTok은 참여가 가치를 만든다는 걸 이해했다."
> — Nii Ahene, Net Influencer

Quibi의 7가지 실패 요인과 FFWD의 대응:

| Quibi 실패 | FFWD 대응 |
|-----------|----------------|
| 소셜 인프라 없음 (공유 불가) | 씬 리액션 + 시네필 커뮤니티 + 소셜 공유 최적화 |
| 커뮤니티 없음 | 시네마 프로필 + 크리에이터 팔로잉 + 토론방 |
| 유료 전용 ($4.99-$7.99) | 프리미엄 + 부분 무료 (웹툰 모델: 3화 무료) |
| 셀러브리티 의존, 사용자 참여 없음 | 이정재 IP + AI 참여 시스템 (캐릭터 대화, 리믹스, 인터랙티브) |
| 콘텐츠 라이선스 (7년 한정) | 자체 IP + 크리에이터 생태계 (영구 콘텐츠 자산) |
| 참여 루프 없음 | 좌뇌·우뇌·오른팔 AI가 **참여를 핵심 루프로 설계** |
| 모바일 전용 → 포기 | 모바일 우선 + 데스크톱/TV 확장 가능 |

**좌뇌·우뇌·오른팔 AI가 바로 Quibi 함정의 탈출구다.**
