# 포털 빌드업 로드맵

> "포털은 만드는 것이 아니라 '되는 것'이다. 순서가 생사를 가른다."

## 전체 타임라인

```
Phase 1: HOOK (0-6개월)       → "매일 여는 이유"
Phase 2: LOOP (6-12개월)      → "시청을 넘어 참여"  
Phase 3: ECOSYSTEM (12-18개월) → "떠날 수 없는 세계"
Phase 4: PORTAL (18-24개월)    → "시네마틱 엔터테인먼트 OS 선언"
```

---

## Phase 1: HOOK — "매일 여는 이유" (0-6개월)

### 목표

사용자가 매일 FFWD를 여는 습관을 만든다. 포털의 전제 조건.

### 핵심 지표

| 지표 | 목표 | 근거 |
|------|------|------|
| DAU/MAU | 40%+ | TikTok 수준의 일상 습관 |
| 세션당 시청 시간 | 15분+ | 숏폼 3-5편 연속 시청 |
| D7 리텐션 | 45%+ | 시장 상위 10% |
| D30 리텐션 | 30%+ | 포털 확장 가능 기준선 |

### 제품 구성

**킬러 오리지널 콘텐츠**
- 이정재 IP 기반 시네마틱 숏폼 시리즈 (3-8분/편, 시즌제)
- 장르: 스릴러/느와르 (이정재 강점) + 로맨스/드라마 (대중성)
- 제작: 전통 촬영 + AI 후반 작업 (비용 70% 절감)
- **매주 2편 이상 신규 에피소드** → 복귀 리듬 형성

**좌뇌 AI v1: 감정 큐레이션**
- 무드/감정 기반 추천 시스템 라이브
- Vionlabs 레퍼런스: 15% 리텐션 증가, 2배 시청량

**일일 시네마틱 모멘트**
- 매일 아침 "오늘의 시네마틱 모멘트" 푸시
- 15초 시네마틱 클립 + 오늘의 무드 추천
- → "매일 여는 첫 번째 이유" (넷플릭스에는 없다)

### 마일스톤

- [ ] 이정재 오리지널 시리즈 시즌 1 런칭 (8편)
- [ ] 좌뇌 AI v1 (무드 큐레이션) 라이브
- [ ] 일일 시네마틱 모멘트 피드 런칭
- [ ] DAU/MAU 40% 달성
- [ ] 누적 다운로드 50만+

## Phase 2: LOOP — "시청을 넘어 참여" (6-12개월)

### 목표

사용자가 "보는 것" 외에 2가지 이상의 행동을 FFWD에서 한다.

### 핵심 지표

| 지표 | 목표 | 근거 |
|------|------|------|
| 시청 외 행동 비율 | 30%+ 사용자 | 포털 전환 임계점 |
| AI 인터랙션 세션 | 일평균 3회+ (활성 사용자) | 참여 습관 |
| UGC(팬 제작 콘텐츠) | 월 1,000편+ | 콘텐츠 플라이휠 시작 |
| 체류 시간 | 25분+ (시청 15 + 참여 10) | 이중 루프 |

### 제품 구성

**오른팔 AI v1: 시네마 안으로 들어가기**
- AI 캐릭터 인터랙션 (최소 3개 캐릭터)
- 인터랙티브 숏폼 시리즈 (스토리 분기)
- 씬 리액션 시스템

**우뇌 AI v1: 나도 시네마를 만든다**
- 시네마틱 리믹스 (스타일 변환)
- 시네마틱 필터 ("이정재 느와르 톤")
- AI 스토리보드 (텍스트 → 컷 구성)

**좌뇌 AI v2: 시네마틱 프로파일**
- 사용자별 시네마 취향 지도
- 개인화 고도화

### 마일스톤

- [ ] AI 캐릭터 인터랙션 런칭
- [ ] 인터랙티브 숏폼 시리즈 1편
- [ ] AI 리믹스 + 시네마틱 필터 런칭
- [ ] 시청 외 행동 비율 30%+
- [ ] 소셜 공유 통한 오가닉 유입 20%+

## Phase 3: ECOSYSTEM — "떠날 수 없는 세계" (12-18개월)

### 목표

FFWD 안에서 **만들고, 보고, 사고, 연결하는** 생태계를 구축한다.

### 핵심 지표

| 지표 | 목표 | 근거 |
|------|------|------|
| 크리에이터 수 | 10,000명+ | 양면 마켓플레이스 임계점 |
| 크리에이터 콘텐츠 비중 | 전체 시청의 30%+ | 콘텐츠 자립 |
| 커머스 전환율 | 시청 → 구매 클릭 2%+ | TikTok Shop 벤치마크 |
| 커뮤니티 MAU | 전체 사용자의 40%+ | 소셜 락인 |

### 제품 구성

**크리에이터 마켓플레이스 (우뇌 AI v2)**
- AI 숏필름 메이커: 텍스트 → 시네마틱 숏필름 (1-3분)
- 3-tier 크리에이터 시스템 (팬/인디/프로)
- 수익화: 시청 배분 + 팬 구독 + 브랜디드 콘텐츠

**Scene Commerce (오른팔 AI v2)**
- 영상 속 상품 자동 태깅 + 즉시 구매
- 촬영지 예약 + 장소 커머스
- TikTok Shop의 시네마틱 프리미엄 버전

**시네필 커뮤니티**
- 에피소드별 토론방
- 시네마 프로필 (취향 지도 공개)
- 감독/배우 Q&A 이벤트

**좌뇌 AI v3: 시장 인텔리전스**
- B2B 트렌드 인사이트 판매
- 콘텐츠 커미셔닝 AI

### 마일스톤

- [ ] AI 숏필름 메이커 런칭
- [ ] 크리에이터 10,000명+
- [ ] Scene Commerce 라이브 (50개+ 브랜드)
- [ ] 시네필 커뮤니티 런칭
- [ ] 크리에이터 콘텐츠 시청 비중 30%+

## Phase 4: PORTAL — "시네마틱 엔터테인먼트 OS" (18-24개월)

### 진입 조건 (모두 충족 시)

- [ ] DAU/MAU 45%+ (3개월 이상 지속)
- [ ] 시청/참여/생성/커머스 중 3가지+ 독립 리텐션 기여
- [ ] 크리에이터 콘텐츠가 전체 시청의 30%+ 차지
- [ ] Scene Commerce 월 거래액 의미 있는 규모

### 포털 확장

**오픈 프로토콜: "시네마 MCP"**
- 외부 서비스가 FFWD에 플러그인으로 연결
  - 음악: 멜론, 스포티파이 → OST 즉시 재생
  - 여행: 야놀자, 에어비앤비 → 촬영지 패키지
  - 패션: 무신사, 29CM → 씬 속 의상 즉시 구매
  - 교육: 영화학과, 연기학원 → 시네마 기법 학습
- 네트워크 효과: 더 많은 플러그인 → 더 많은 사용자 → 더 많은 플러그인

**프로 크리에이터 스튜디오**
- AI 도구 + 펀딩 + 배급을 한 곳에서
- 팬 크라우드펀딩: "이 숏필름 시리즈에 투자하세요"
- FFWD 오리지널 승격 경로

**글로벌 확장**
- K-시네마 숏폼 → 아시아 → 글로벌
- 이정재 글로벌 인지도 (오징어게임) → 영어권 교두보
- AI 더빙/자막 (감정 톤까지 유지)

### 마일스톤

- [ ] "시네마틱 엔터테인먼트 OS" 공식 발표
- [ ] 오픈 프로토콜 파트너 10개+
- [ ] 글로벌 1개 시장+ 진출 (일본 또는 동남아)
- [ ] 연간 반복 매출(ARR) 의미 있는 규모
