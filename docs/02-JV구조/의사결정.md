# 의사결정 & 수익환원

> "Your Vision. Your Frame."

## 의사결정 구조

```
[이사회 (Board)]
 ├── 강정모 (의장, 대표이사)
 ├── 이정재 (Founding Curator)
 ├── 독립이사 1인 (영화계)
 └── 독립이사 1인 (기술/투자)

[경영진]
 ├── 대표이사: 강정모 (키넥션) — 경영·기술·사업 총괄
 ├── CPO (콘텐츠 총괄): 이정재 측 추천
 ├── CTO (기술 총괄): 키넥션 측
 └── COO (운영 총괄): 공동 채용
```

> 정렬 원칙: 모든 주요 의사결정에서 "이 결정이 헌장 6대 원칙 중 어떤 것에 기여하는가?"를 명시적으로 확인한다.

## 수익 환원 구조: 구조적 커밋먼트

Anthropic 창업자들의 80% 기부 서약에서 영감을 받아:

> **FFWD 서약**
>
> 플랫폼 **영업이익의 30%**를 매년 '차세대 창작자 기금'에 재투자

이것은 CSR이 아니라 브랜드 작동 도구다. 약속을 하고, 지키고, 증명하는 것이 반복될 때 "FFWD는 진짜다"라는 시장 신뢰가 형성된다.

- 이 기금은 시상식 제작지원비(30팀 × 3억) + 창작자 교육 프로그램에 사용
- 정관에 명시하여 법적 구속력 부여
- Anthropic의 "80% 서약"처럼, 창업자의 진정성을 보여주는 구조적 커밋먼트

## 수수료 배분 상세

```
[유료 열람 결제 ₩400]
 ├── 결제 수수료 (PG) → 3.5% = ₩14
 ├── 창작자 → 66.5% = ₩266
 └── 플랫폼 → 30% = ₩120
      ├── 서버/인프라 → 10%
      ├── 콘텐츠 큐레이션 → 5%
      ├── 차세대 기금 → 9% (영업이익의 30% 상당)
      └── 영업이익 → 6%
```
