## LTV (고객 생애 가치) 분석

### 가정

| 항목 | 보수적 | 낙관적 |
|------|--------|--------|
| 월 ARPU (무료 사용자, 광고) | ₩200 | ₩500 |
| 월 ARPU (유료 구독자) | ₩4,900 | ₩4,900 |
| 유료 전환율 | 3% | 7% |
| 월 이탈률 (Churn) | 15% | 8% |
| 평균 수명 | 6.7개월 | 12.5개월 |

### LTV 계산

**보수적 시나리오:**
- Blended ARPU = ₩200 × 0.97 + ₩4,900 × 0.03 = ₩341/월
- 평균 수명 = 1/0.15 = 6.7개월
- **LTV = ₩341 × 6.7 = ₩2,285**
- **LTV/CAC = 1.0** ← ⚠️ 거의 손익분기

**낙관적 시나리오:**
- Blended ARPU = ₩500 × 0.93 + ₩4,900 × 0.07 = ₩808/월
- 평균 수명 = 1/0.08 = 12.5개월
- **LTV = ₩808 × 12.5 = ₩10,100**
- **LTV/CAC = 4.4** ← ✅ 건강한 비율

### 벤치마크

| 플랫폼 | ARPU | LTV/CAC |
|--------|------|---------|
| Netflix 한국 | $84.38/년 (~₩7,000/월) | ~3.5 |
| MUBI 글로벌 | ~$9.17/월 (~₩12,000) | ~3.0 |
| Spotify | ~₩7,000/월 | ~2.5 |
| **FFWD (보수적)** | **₩341/월** | **~1.0** |
| **FFWD (낙관적)** | **₩808/월** | **~4.4** |

::: warning ⚠️ 핵심 과제
보수적 시나리오에서 LTV/CAC ≈ 1.0은 **지속 불가능**합니다. 유료 전환율을 **최소 5%** 이상으로 끌어올리거나, 광고 ARPU를 ₩500 이상으로 높여야 사업이 성립합니다.
:::

## 에피소드 단위 경제

### 에피소드당 수익 (₩400 과금 기준)

| 항목 | 금액 | 비율 |
|------|------|------|
| PG 수수료 | ₩14 | 3.5% |
| 창작자 몫 | ₩266 | 66.5% |
| 플랫폼 몫 | ₩120 | 30% |
| └ 서버/인프라 | ₩40 | 10% |
| └ 큐레이션 | ₩20 | 5% |
| └ 차세대 기금 | ₩36 | 9% |
| └ 영업이익 | ₩24 | 6% |

**플랫폼 순이익 ₩24/에피소드** — 월 인건비 ₩7억을 커버하려면 **월 2,917만 결제 건이 필요**. MAU 10,000 기준 사용자당 월 2,917건은 현실적이지 않음.

→ **에피소드별 과금만으로는 사업 성립 불가. 구독 모델 + 광고가 핵심 수익원이어야 함.**

## 유저 획득 퍼널

```
[인지] 유튜버 콜라보 + SNS + PR
    ↓ (CTR 2~5%)
[방문] 웹사이트/앱 방문
    ↓ (회원가입 전환 30%)
[가입] 무료 회원가입
    ↓ (첫 시청 80%)
[시청] 무료 3화 시청
    ↓ (유료 전환 3~7%)
[과금] 유료 구독 또는 에피소드 구매
    ↓ (재결제 50~70%)
[충성] 월간 활성 유료 사용자
```

| 단계 | 보수적 전환율 | Year 1 수치 |
|------|-------------|------------|
| 노출 | — | 500만~1,000만 |
| 방문 | 3% | 150,000~300,000 |
| 가입 | 30% | 45,000~90,000 |
| 첫 시청 | 80% | 36,000~72,000 |
| 유료 전환 | 3% | 1,080~2,160 |
| 재결제 | 60% | 648~1,296 |
