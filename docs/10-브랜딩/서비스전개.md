# 서비스 프로그램

사용자가 처한 상황(시나리오)을 중심으로 서비스를 설계한다 — 기능이 아니라 맥락으로 분류한다.

## 크리에이터 프로그램

| Program | Tagline | 시나리오 |
|---|---|---|
| DEBUT | "Your First Frame Matters." | 첫 작품을 업로드하는 신인 크리에이터 |
| SIGNATURE | "Make Every Frame Unmistakably Yours." | 자기 스타일을 확립하려는 중견 크리에이터 |
| MASTERCLASS | "Teach What Only You Can." | 마스터가 기법을 전수하는 교육 콘텐츠 |
| COMMISSION | "Your Vision, Our Investment." | 플랫폼이 직접 투자하는 제작 위탁 |
| COLLAB | "When Frames Collide." | 크리에이터 간 협업 프로젝트 |

### DEBUT — 신인 발굴

- 전국 200여 개 연극영화과 졸업작품이 FFWD의 핵심 소스
- 첫 업로드 시 전담 큐레이터 매칭
- "First Frame" 배지 — 처음 올린 크리에이터를 위한 특별 노출

### SIGNATURE — 스타일 확립

- 3편 이상 업로드 + 일정 조회수 달성 시 SIGNATURE 크리에이터 전환
- 개인 프로필 페이지, 팬 구독 기능
- 브랜드 콜라보 매칭 우선권

### COMMISSION — 투자 제작

- FFWD가 직접 제작비를 지원하는 오리지널 콘텐츠
- 이정재 Founding Curator의 직접 셀렉션
- 분기별 10편 선정, 편당 최대 5,000만 원 지원

## 관객 프로그램

| Program | Tagline | 시나리오 |
|---|---|---|
| CURATED | "Selected, Not Suggested." | 알고리즘이 아닌 큐레이터가 선별 |
| FESTIVAL | "The Festival Never Ends." | 온라인 영화제 상설 운영 |
| DEEP DIVE | "Behind Every Frame." | 메이킹 필름, 감독 인터뷰, 비하인드 |
| CIRCLE | "Watch Together. See More." | 관객 커뮤니티, 토론, 투표 |

### CURATED — 큐레이션

- 매주 "Weekly Selection" — 큐레이터가 직접 고른 5편
- 월간 테마 컬렉션 (예: "빛의 언어", "침묵의 연기", "도시의 소리")
- 알고리즘 추천은 보조 도구, 큐레이션이 메인

### FESTIVAL — 상설 영화제

- 연 4회 온라인 FFWD Film Festival
- 관객 투표 + 전문가 심사 병행
- 수상작에 추가 제작지원금 + 글로벌 노출

## 론칭 로드맵

### Phase 1: ORIGIN (0-6개월) — 카테고리 창조

"Build the Next Hottest Thing."

- 이정재를 Founding Curator로 포지셔닝
- 한국 주요 감독 10인의 "첫 FFWD 작품" 동시 공개
- 칸/부산 영화제와의 공식 숏폼 파트너십 추진
- 핵심 메트릭: 크리에이터 100명, 작품 500편, 월간 관객 50만

### Phase 2: MOMENTUM (6-18개월) — 깊이 확장

"Go From Growing Pains to High Performance."

- 아시아 영화인 네트워크 확장 (일본, 대만, 인도)
- 크리에이터 수익 모델 본격화 (구독 + 개별 작품 유료)
- 첫 FFWD Awards 시상식
- 핵심 메트릭: 크리에이터 1,000명, 월간 관객 500만

### Phase 3: AUTHORITY (18-36개월) — 프리미엄화 + 글로벌 확장

"Elevate Every Signal." + "Enter New Markets with an Edge."

- 헐리우드/유럽 영화인 유치
- 브랜드 콜라보레이션 (럭셔리 브랜드 × FFWD 오리지널)
- IP 라이센싱 (FFWD 오리지널 → 장편 영화화)
- 핵심 메트릭: 글로벌 크리에이터 5,000명, 월간 관객 2,000만
