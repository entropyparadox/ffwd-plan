# 브랜드 시스템 (Augments)

로고와 색상을 넘어, 브랜드가 세상에서 작동하는 방식 자체를 시스템화한다.

## Augments란?

전통 브랜딩은 "로고 + 컬러 + 서체"로 끝난다. FFWD는 그 너머를 설계한다. 브랜드가 다양한 터치포인트에서 일관되고 강력하게 작동하도록 만드는 **브랜드 성능 확장 컴포넌트** — 이것을 Augments라 부른다.

## 시각 시스템

| Augment | FFWD 적용 |
|---|---|
| Custom Typeface | 영화 크레딧에서 영감받은 세리프 + 산세리프 이중 서체 시스템. 크레딧의 클래식함 + UI의 현대적 가독성 |
| Motion Signature | 필름 게이트가 열리는 듯한 트랜지션 — 모든 영상 시작 시 적용. 빛이 프레임을 채우는 0.5초 |
| Memory Marker | "✦" 심볼 — 필름 프레임 형태의 로고. 영상 우측 하단 워터마크로 존재감 |
| Colorgraph | 딥 블랙(#0a0a0a) + 시네마 골드(#c8a96e) + 필름 그레인 텍스처 |
| Shelf Pop | 앱스토어/SNS에서의 즉각적 시각 차별화 — 골드 악센트가 다른 숏폼 앱과 구별 |

### Colorgraph 상세

```
Primary:   #0a0a0a  딥 블랙 — 영화관의 어둠
Accent:    #c8a96e  시네마 골드 — 필름의 따뜻함, 시상식의 격조
Surface:   #1a1a2e  미드나이트 — 인터페이스 배경
Text:      #f5f5f5  소프트 화이트 — 스크린의 빛
Muted:     rgba(255,255,255,0.5)  — 보조 텍스트
```

## 언어 시스템

| Augment | FFWD 적용 |
|---|---|
| Language Bridge | 영화 전문용어 ↔ 일반 관객 언어 자동 번역 (UI 내). "시네마토그래피" → "카메라의 눈", "미장센" → "장면 구성" |
| Verbal System | 조용한 확신 톤, "프레임/빛/장면" 어휘 체계. 동사로 시작하는 선언적 카피 |
| Sticky Dissonance | "3분짜리 영화" — "짧은" + "영화"의 의도적 불일치가 뇌에 각인 |

### Language Bridge 실습

| Before (전문 언어) | After (FFWD Bridge) |
|---|---|
| "DCI-P3 색 공간에서 그레이딩된 작품" | "영화관에서 보는 것과 같은 색감" |
| "180도 셔터 앵글 기반 모션 블러" | "영화처럼 자연스러운 움직임" |
| "숏/리버스숏 대화 장면" | "두 사람이 마주 보는 클래식 구도" |

## 전략 시스템

| Augment | FFWD 적용 |
|---|---|
| Heritage Capsule | K-Cinema의 유산 (봉준호, 박찬욱, 이정재)을 현대적으로 캡슐화. "오래됨 = 검증됨"이 아니라 "시작됨 = 혁신의 기원" |
| High Expressive Ratio | 한 장의 스틸컷으로 작품의 전체 분위기를 전달하는 썸네일 시스템. 최소한의 정보 → 최대한의 감성 |
| Culture Sync Logic | 각 국가의 영화 문화와 동기화된 로컬 큐레이션. 한국은 학교 졸업작 중심, 일본은 자주영화 중심 등 |
| Endurance Function | 5년 후에도 유효한 브랜드 시스템. 유행을 쫓지 않는 클래식한 비주얼 |
| 3P IP Adapter | 외부 IP(기존 영화/드라마 원작)를 FFWD 숏폼으로 재해석하는 프레임워크 |

## Augments 적용 체크리스트

- [ ] 브랜드가 최소 3개 이상의 터치포인트(앱, SNS, 영화제)에서 일관되게 작동하는가?
- [ ] 기술 언어와 소비자 언어 사이의 번역 체계(Language Bridge)가 있는가?
- [ ] 기억에 남는 순간(Motion Signature)이 설계되어 있는가?
- [ ] 앱스토어/SNS에서의 즉각적 차별화(Shelf Pop)가 가능한가?
- [ ] 브랜드 시스템이 5년 후에도 유효한가? (Endurance Function)
- [ ] 새로운 프로그램 추가 시 자연스럽게 확장 가능한가?
